# Markdown Guide

This guide shows you how to use Markdown to write beautiful documentation.

## Basic Syntax

### Headers

```md
# H1 Header
## H2 Header
### H3 Header
```

### Emphasis

```md
*italic* or _italic_
**bold** or __bold__
***bold italic*** or ___bold italic___
```

### Lists

```md
1. First item
2. Second item
   - Subitem
   - Another subitem
```

## Code Blocks

```javascript
function example() {
  console.log('Hello, World!');
}
```
