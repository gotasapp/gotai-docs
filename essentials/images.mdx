# Working with Images

Learn how to effectively use images in your documentation.

## Basic Image Syntax

```md
![Alt text](/path/to/image.png)
```

## Best Practices

1. Use descriptive alt text
2. Optimize images for web
3. Consider responsive images
4. Use appropriate image formats
   - PNG for graphics
   - JPG for photos
   - SVG for vectors
